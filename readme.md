# Casos de intento de suicidio y suicidio consumado en Chile

Procesamiento y análisis de datos de suicidios, obtenidos desde el Ministerio de Salud de Chile, en base al egreso con o sin vida de cada paciente.

- Egresos de pacientes por intento de suicidio
- Egresos de pacientes fallecidos por suicidio consumado

### Fuente 
- Ministerio de Salud, Egresos hospitalarios
- Datos abiertos, Departamento de Estadísticas e Información de Salud: https://deis.minsal.cl/#datosabiertos
